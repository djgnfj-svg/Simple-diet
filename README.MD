## 🍱소개

시골에서는 식단을 제공해주는 서비스가 지원되지 않아서 만든 프로젝트입니다.  

간단식단은 자신의 신체정보와 활동정보를 입력하면 대사량을 계산해 알맞은 식단과 식품의 링크를 보내주는 프로그램입니다.

<br>

---

## 🛠사용기술 및 도구
- Backend : Python, Django, AWS-EC2, AWS-RDS
- Frontend : React
- Dagabase : Mysql
